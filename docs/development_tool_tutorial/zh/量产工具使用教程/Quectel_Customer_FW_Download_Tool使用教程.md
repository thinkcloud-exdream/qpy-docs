## Quectel_Customer_FW_Download_Tool

支持型号：EC200U系列、EC600U-CN、EC600G、EC800G、EC600E。

（1）点击“**Load FW Files**”按钮，加载量产固件，需要手动选择AT端口，点击Start或Start All开始升级。

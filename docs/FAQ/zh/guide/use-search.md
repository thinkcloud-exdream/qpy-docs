# 问题查找

此指南目前有以下两个部分：

- 搜索问题技巧
- 问题分类框架

## 搜索问题技巧

使用界面右上角的搜索框是您获取问题答案的最快途径。

搜索框支持多个关键词使用空格分隔，将问题中的关键词提取出来并搜索它们，此时搜索结果会得到最匹配的结果。

比如问题为：`CAT1的下行带宽是多少？`

此时搜索：`CAT1`、`下行带宽` 等关键字为宜。
